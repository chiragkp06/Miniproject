About PneumoScan

PneumoScan is a cloud-enabled AI tool designed for academic research and rapid clinical screening. It demonstrates the power of computer vision in diagnosing pneumonia through X-ray analysis using Flask backend and cloud tools (like AWS Rekognition).

This tool is not meant for real-time medical diagnosis but can serve as a prototype for future healthcare innovation.

Features

Fast Detection

Upload and analyze chest X-rays within seconds using a clean, fast interface.

AI Powered

Utilizes cloud-based AI models to detect and highlight pneumonia signs.

Easy Integration

Designed to integrate with small clinics and hospitals for testing use cases.

FAQs

What kind of images can I upload?

JPG, JPEG, PNG formats of chest X-rays only.

Is this tool certified for diagnosis?

No, it is strictly for educational and research use, not clinical decisions.

Do I need to sign up?

Yes, only registered users can access the detection tool after logging in.
